公开符号

Symbol.iterator

  • 为任意对象定义自己的迭代器逻辑，会覆盖默认的迭代器。

Symbol.toString

  • 指定在[object ___]字符串化时使用的字符串值。

Symbol.hasInstance

  • 构造器函数上的一个方法，接受实例对象值，通过返回true/false来指示这个值是否可以被认为是一个实例。
  • 要设置@@hasInstance，必须使用Object.definedProperty(..)

Symbol.species

  • @@species，控制要生成实例时，类的内置方法使用哪一个构造器。

Symbol.toPrimitive

  • 对象为了某个操作（比如比较==或者相加+）必须被强制转换为一个原生类型值的时候。
  • 调用ToPrimitive的运算期望的类型，会提供一个提示（hint）指定“string”、“number”、”default”；加法+没有提示（传入“default”），而乘法*提示为“number”，String（arr）提示为“string”。

Symbol.match

Symbol.replace

Symbol.search

Symbol.split

Symbol.isConcatSpreadable

  • 用来指定如果把它传给一个数组的concat(..)是否应该将其展开。（布尔值）

Symbol.unscopables

  • 指示使用with语句时哪些属性可以或不可以暴露为词法变量。

代理

  • 代理是一种你创建的特殊的对象，它“封装”另一个普通对象—或者说档在这个普通对象的前面。

  • 可取消代理用Proxy.revocable，接收target和handlers，返回proxy和revode的对象。

  • 使用代理

    • 代理在先，代理在后。

  • Reflect API

    • 属性排序

      • 1、首先，按照数字上升排序，枚举所有整数索引拥有的属性
      • 2、然后，按照创建顺序枚举其余的拥有的字符串属性名
      • 3、最后，按照创建顺序枚举拥有的符号属性
  • Reflect.ownkeys、Object.getOwnPropertyNames、Object.getOwnPropertySymbols 遵循属性排序规则

尾递归调用

  • 尾调用的函数调用模式，可以避免额外栈帧分配的方式进行优化
  • 非TCO优化，其中一种叫做trampolining技术，相当于把每个部分结果用一个函数表示，这些函数或者返回另外一个部分结果函数，或者返回最终结果。
