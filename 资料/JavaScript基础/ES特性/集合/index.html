Map

  • new Map([[“a”, “foo”], [“b”, “bar”]])

  • API

    • set(..)

    • get(..)

    • delete(..)

    • has(..)

    • clear()

    • size

  • Map值 values(…)；Map键 keys(…)

  • map的本质是允许你把某些额外的信息（值）关联到一个对象（键）上，而无需把这个信息放入对象本身（可以用对象作为键）

  • 如果用键作为对象，后来对象被丢弃（所有引用解除），map本身仍旧会保持其项目

WeakMap

  • 只接受对象作为键。

  • 键作为对象，后来对象被丢弃（所有引用解除），WeakMap中这个项目也会被移除。

  • API

    • set(..)

    • get(..)

    • delete(..)

    • has(..)

  • WeakMap只是弱持有它的键

Set

  • set是一个值的集合，其中的值唯一；唯一性不允许强制转换，所以1和“1”是不同的值

  • API

    • add(…)

    • delete(..)

    • clear()

    • size

  • new Set([1,2])

  • has不区分-0与0

WeakSet

  • WeakSet对其值也是弱持有

TypedArray

  • Int8Array：8 位有符号整数，长度 1 个字节。

  • Uint8Array：8 位无符号整数，长度 1 个字节。

  • Uint8ClampedArray：8 位无符号整数，长度 1 个字节，溢出处理不同。

  • Int16Array：16 位有符号整数，长度 2 个字节。

  • Uint16Array：16 位无符号整数，长度 2 个字节。

  • Int32Array：32 位有符号整数，长度 4 个字节。

  • Uint32Array：32 位无符号整数，长度 4 个字节。

  • Float32Array：32 位浮点数，长度 4 个字节。

  • Float64Array：64 位浮点数，长度 8 个字节。
