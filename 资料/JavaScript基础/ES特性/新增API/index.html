Array

静态函数Array.of替代Array()

  • Array()如果只传入一个数值返回一个空数值，传入的值会是length属性

静态函数Array.from(..)实例化类（似）数组对象

  • 避免空槽位

    • Array.from({ length: 4 })

  • 映射

    • 把来自于源的每个值映射/转换到返回值

  • 参数1：类数组对象
    参数2：回调函数
    参数3：回调函数this绑定

原型方法copyWithin(..)

  • 从一个数组复制一部分到同一个数组的另一个位置，覆盖这个位置原有的值。

  • 任何参数为负值，就是当作相对于数组的结束的相对值

    // 任何参数为负值，就是当作相对于数组的结束的相对值

    // 4/-2替换1，5/-替换2；[4, 5, 3, 4, 5]

    [1, 2, 3, 4, 5].copyWithin(0, -2)

    // 从-2开始复制到-1结束,4/-2替换1；[4, 2, 3, 4, 5]

    [1, 2, 3, 4, 5].copyWithin(0, -2, -1)

原型方法fill(.. )

  • 指定值完全（或部分）填充已存在的数组

原型方法find(..)

  • 回调返回true/真值，还会返回实际的数组值

原型方法findIndex(..)

  • 可以通过回调返回自定义匹配的索引值

原型方法entries()、values()、keys()

Object

  • NaN或者-0/0值的比较，使用Object.is

静态函数Object.getOwnPropertySymbols(..)

  • 获取对象的所有符号属性

静态函数Object.setPrototypeOf(..)

  • 设置对象的[[Prototype]]用于行为委托

静态函数Object.assign(..)

  • 不可枚举的属性和非自有的属性都被排除在赋值过程之外
